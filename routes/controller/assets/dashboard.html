Welcome to AgentOpt
Review your AI usage history, get analysis-driven configuration suggestions, and approve changes before they reach your local workspace.
Install the CLI
Install the released CLI on the machine you want to connect. The release build does not require Go, and the installer provisions Node.js automatically when needed.
Install command
curl -fsSL https://raw.githubusercontent.com/Royaltyprogram/aiops/main/scripts/install.sh | sh
Authenticate your CLI
Create a token here, then run the login command in your terminal.
Your CLI token
Click "Create CLI token" above to generate one.
Login command
agentopt login --server http://127.0.0.1:8082
Connect a workspace
Link a local repository so AgentOpt can start analyzing your workflow.
Connect command
agentopt connect --repo-path .
